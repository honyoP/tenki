# Keyboard Shortcuts

Tenki supports vim-style navigation.

## Navigation

| Key | Action |
|-----|--------|
| `j` / `Down` | Move down |
| `k` / `Up` | Move up |
| `Tab` | Next pane |
| `Shift+Tab` | Previous pane |
| `Enter` | Select / Follow link |

## Actions

| Key | Action |
|-----|--------|
| `e` | Edit in $EDITOR |
| `r` | Refresh |
| `q` | Quit |

See [[Welcome to Tenki]] for more information.
