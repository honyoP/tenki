# Markdown Syntax

Tenki supports standard markdown syntax.

## Headers

Use `#` for headers:
- `# H1`
- `## H2`
- `### H3`

## Links

- Standard markdown links: `[text](url)`
- [[Wikilinks]]: `[[Note Name]]`

## Lists

Both ordered and unordered lists are supported.

See also: [[Welcome to Tenki]]
